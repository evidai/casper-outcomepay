OutcomePay

Autonomous agents pay for real-world work on Casper — but the money is escrowed and released only when the outcome is machine-verified, under a hard on-chain budget cap the agent can't exceed. Every settlement is a verifiable on-chain receipt.

🟢 Live on Casper Testnet 🔒 On-chain budget cap ✅ Settle on verified outcome ↩︎ Refund on failure — green-or-no-fee 🤖 MCP-native · x402
How it works
1

Lock

The agent escrows CSPR for a job. The contract refuses if it would exceed the agent's on-chain cap — overspend is impossible.

2

Verify

A machine check decides the outcome — here: dependency CVEs cleared and tests green. Binary, not subjective.

3

Settle / Refund

Pass → funds released to the provider with a receipt. Fail → refunded to the payer. The chain decides, not the agent.

agent→ lock (cap-checked) → verify → settle or refund → on-chain receipt
