# Contributing

Thanks for your interest in OutcomePay!

- **Bugs / ideas:** open a GitHub Issue with steps to reproduce or a clear description.
- **Pull requests:** fork, create a feature branch, keep changes focused, and describe
  what/why in the PR body. Make sure `contract/` builds and existing flows
  (lock → verify → settle/refund) still pass before submitting.
- **Questions:** GitHub Discussions/Issues, or the Casper dev community
  (Telegram: https://t.me/CSPRDevelopers).
